B) How to Use - top

First Please Register, at http://license.zincksoft.com/ with your CodeCanyon purchase code. After Registration, Please login to get Activation Key for your Installation. If you wanted to use this panel for more than one Server you can Add More IP Registartion from this Panel for one time fee for $10 per IP

After that, Extract all the files form the zip. The file you need to run is install.sh as you run the this file it will ask Serial Key and some details from you after that it will get all the software packages and install everything smoothly in your server. after installation is done, your server will be restarted and you can browse your control panel at your server IP:786, default username is administrator and password is 123456789 you can change it after your first login.

Installation Process :

  • Unzip the File that you get after Purchase.
  • Get Install.sh file fron the folder.
  • Access your VPS or Dedicated server via WinSCP or any SFTP Client with Root Access
  • Upload install.sh
  • Change Permission of install.sh to 0777
  • Now Access your VPS or Dedicated server via Putty with Root Access
  • Go to folder where you uploaded install.sh and Run install.sh with this command "bash install.sh"
  • As the install starts, it will ask for timezone and confirm server, after that it will install everything and in the end ask for restart.
  • You can access your Conrtol Panel via your Server IP and Port no 786 for example : http://185.81.166.71:786

Thats it ! you are good to go to manage your server now.
